# AI News & Resource Site

一个基于 Vue + Node.js + MongoDB 的 AI 最新资讯与资源下载网站。

## 目录结构

```
ai-news-resource-site/
  backend/    # Node.js + Express + MongoDB 后端
  frontend/   # Vue 3 + Vite 前端
```

## 启动方法

### 1. 后端
```bash
cd backend
npm install
node index.js
```

### 2. 前端
```bash
cd frontend
npm install
npm run dev
```

## 依赖
- 前端：Vue 3, Vite
- 后端：Node.js, Express, Mongoose, CORS
- 数据库：MongoDB

## 功能规划
- AI 最新资讯展示
- AI 资源下载（如论文、工具包、数据集等）
- 后台管理（可选）
- 响应式简洁科技感界面
